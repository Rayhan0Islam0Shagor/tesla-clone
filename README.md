## Installation

Install my-project with npm

```bash
  git clone "https://github.com/Rayhan0Islam0Shagor/tesla-clone.git"
  npm install
  npm start
```

Install my-project with yarn

```bash
  git clone "https://github.com/Rayhan0Islam0Shagor/tesla-clone.git"
  yarn
  yarn start
```
